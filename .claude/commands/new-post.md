새 블로그 글을 작성하기 위한 마크다운 파일을 생성해줘.

다음 순서로 진행해:

1. 사용자에게 아래 정보를 질문해:
   - 글 제목 (한국어 가능)
   - 태그 (쉼표로 구분, 예: Java, Spring Boot)
   - 한 줄 요약 (summary)

2. 제목을 기반으로 slug를 만들어:
   - 한국어는 영어로 변환 (예: "JPA 성능 최적화" → "jpa-performance")
   - 소문자, 단어 사이는 하이픈(-)
   - 특수문자 제거

3. 오늘 날짜를 YYYY-MM-DD 형식으로 사용해.

4. `src/posts/{slug}.md` 파일을 아래 형식으로 생성해:

```
---
title: "{제목}"
date: "{오늘날짜}"
tags: [{태그 배열}]
summary: "{요약}"
---

## 시작하며

(본문을 여기서부터 작성하세요)
```

5. 파일 생성 후 `/blog/{slug}` 경로로 접근 가능하다고 알려줘.
