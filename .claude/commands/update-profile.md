src/constants/profile.ts 의 프로필 정보와 스킬을 업데이트해줘.

다음 순서로 진행해:

1. `src/constants/profile.ts` 파일을 읽어서 현재 PROFILE과 SKILLS 내용을 보여줘.

2. 사용자에게 수정할 항목을 선택하게 해:
   - 기본 정보 (이름, 직무, 한 줄 소개, 이메일, GitHub)
   - Backend 스킬 목록
   - DevOps & Tools 스킬 목록

3. 선택한 항목의 현재 값을 보여주고 새 값을 입력받아.

4. 스킬 추가 시 색상 클래스도 함께 지정해:
   - 형식: `bg-{색상}-500/20 text-{색상}-300 border-{색상}-500/30`
   - 기존 스킬 색상 패턴을 참고해서 어울리는 색상 제안

5. 수정 내용을 `src/constants/profile.ts` 에 반영하고 변경된 부분을 요약해줘.
